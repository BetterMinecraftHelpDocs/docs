# BetterMinecraftHelpDocs

这是 更好的我的世界文档的文档源代码

## Join us
fork 这个仓库  
然后输入下方的命令  
```sh
git clone forked-repo(在你fork的仓库里面有对应的git地址)
cd docs
npm i
npm run docs:dev
```
就可以开始编辑文档并且查看变化了 

请先查看
